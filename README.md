## 사용한 기술

- react
- react-map-gl
- deckgl
- react-spring
- d3
- material-ui

## 설치 및 실행

```bash
git clone git@github.com:sangkukbae/seoul-public-bicycle.git
```

```bash
npm install
```

```bash
npm start
```
